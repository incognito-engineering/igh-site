# igh-site

Official static website of [igh.inc](https://igh.inc) deployed via [Cloudflare Pages](https://pages.cloudflare.com/).

## Development

Clone this repository and start the local dev server:

```sh
python3 -m http.server 8000
```

And visit `http://localhost:8000` to preview changes.

## Deployment

[Cloudflare Pages](https://pages.cloudflare.com/) serves the repository root.

```text
Framework preset: None
Build command: none
Build output directory: /
Production branch: master
```
